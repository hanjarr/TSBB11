GIT SETUP INSTRUCTIONS

1. Skapa ett konto på GitHub, bli inbjuden till repository TSBB11
2. Skapa en mapp på valfritt ställe 
3. $ git init (i den nya mappen)
4. $ git config user.name "NAME" (samma användarnamn som till github)
5. $ git config user.email "EMAIL" (samma mail som till github)
6. $ git remote add origin https://github.com/hanjarr/TSBB11

För att hämta ner filer
1. $ git pull origin master

För att lägga upp filer
1. $ git add FILENAME
2. $ git commit -m "COMMENT"
3. $ git push origin master
